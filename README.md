# 로아 검문소
> 로스트아크 군장검사를 OCR 기능을 이용하여 자동화한 웹입니다.


![](../header.png)

## 개발 환경 실행 방법
Node v20.11.0
```sh
npm install
npm run dev
```

## 사용 예제

스크린 샷과 코드 예제를 통해 사용 방법을 자세히 설명합니다.

## 사용된 기술
* TypeScript
* React v18
* Next.js v14
* Tesseract.js v5
* Tanstack Query v5
* Zustand v4
* Shadcn/UI
* Tailwind CSS
